Coursera John Hopkins Practical Machine Learning Course Project

Qualitative Assessment of Weight Lifting Exercises with Machine Learning

Download this project as a .zip file Download this project as a tar.gz file

The course assignment for the Practical Machine Learning course offered by John Hopkins University on the Coursera platform was an exercise in applying a number of machine learning techniques to build and evaluate model capable of predicting the activity class for controlled variations of a Unilateral Dumbbell Biceps Curl exercise conducted by 6 human subjects.

The analysis was conducted using RStudio and published to GitHub.
